# Smart Health & Medical History Tracker

## 📌 Project Description
This project is a Python-based system used to store, manage, and analyze personal health records. It helps in tracking patient details, vital signs, and medical history in a simple and organized way.

## 🚀 Features
- Add and manage patient records
- Track vital signs (BP, sugar, etc.)
- Search patient data
- File storage using JSON and CSV
- Data analysis using NumPy and Pandas
- Data visualization using Matplotlib

## 🛠 Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib

## 📂 Project Structure
- Week 1–4: Python Basics and Functions
- Week 5–7: Data Structures and OOP
- Week 8–9: File Handling and Modules
- Week 10–12: Data Analysis and Visualization
